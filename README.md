# Audio Transcriber

Локальный сервис транскрибации аудио и голосовых сообщений с интерфейсом на Gradio.

## Что умеет

- Загружать один или несколько файлов
- Поддерживать форматы `.mp3`, `.wav`, `.m4a`, `.ogg`, `.opus`, `.aac`, `.mp4`, `.mpeg`, `.webm`
- Распознавать речь через `faster-whisper`
- Сохранять текстовые результаты в `.txt`
- Добавлять таймкоды
- Останавливать обработку с сохранением текущего прогресса

## Требования

- macOS
- Python 3.11+
- `ffmpeg`

## Установка

Установите `ffmpeg`:

```bash
brew install ffmpeg
```

Создайте и активируйте виртуальное окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python -B Транскрибатор.py
```

После запуска сервис будет доступен в браузере по локальному адресу, например:

```text
http://127.0.0.1:7860
```

## Как пользоваться

1. Откройте локальный адрес в браузере.
2. Загрузите один или несколько файлов.
3. Выберите язык, режим результата и профиль скорости.
4. Нажмите `Транскрибировать`.
5. При необходимости нажмите `Остановить`, чтобы сохранить уже готовый прогресс.

## Структура проекта

- `Транскрибатор.py` — основное приложение
- `requirements.txt` — зависимости Python
- `transcription_outputs/` — результаты транскрибации
